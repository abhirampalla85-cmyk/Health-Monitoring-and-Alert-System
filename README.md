# Health Monitoring and Alert System

## Description

A full-stack health monitoring system that tracks user health data and generates alerts for abnormal conditions.

## Tech Stack

* Java (Spring Boot)
* HTML, CSS, JavaScript
* Maven

## Features

* User authentication (login/signup)
* Health data tracking
* Alert system
* Dashboard interface

## Folder Structure

* backend/monitoring → Backend (Spring Boot)
* website → Frontend (HTML/CSS/JS)

## How to Run

1. Run backend (Spring Boot application)
2. Open frontend files in browser
